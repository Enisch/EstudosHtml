Uma lista <ol> é ordenada numericamente <ul> enquanto que esta não tem ordenação.
ambas possuem dentro de seu corpo elementos <li> que são items de lista.
listas ol podem usar o parametro type que define seu estilo de ordenação ex: número,letras e I e o start que define onde inicia a ordenação, basta inserir o número, caso queira a letra e,por exemplo, inisira no start 5.
Listas ul também tem type, porém só três, disc, padrão, circle e square.
